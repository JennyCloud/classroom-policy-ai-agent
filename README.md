# Classroom Policy AI Agent 🤖

## 🎯 Project Goal
A proof-of-concept AI agent built to assist **Education Assistants (EAs)** in British Columbia. Unlike a standard chatbot, this agent uses **Retrieval-Augmented Generation (RAG)** to provide answers grounded in official provincial standards and automates administrative tasks like drafting student observation reports.

## 🛠️ Technology Stack
* **Platform:** Microsoft Copilot Studio (Agentic AI)
* **Knowledge Base:** [BC Teachers' Federation (BCTF)](https://bctf.ca/) live web data
* **Logic Framework:** Natural Language Understanding (NLU) with custom Topic Triggering

## 🚀 Key Features & Skills
* **Policy Specialist:** Summarizes complex school policies into actionable bullet points for EAs.
* **Observation Reporter:** A custom workflow that collects student incident data and generates a professional, BCTF-compliant email draft for classroom teachers.
* **Role-Awareness:** Programmed to recognize the boundaries of the EA role, ensuring all advice remains within professional guidelines.

## 📂 Repository Contents

## 📝 How to Use
1. Ask the agent about BC educational professional standards or inclusivity policies.
2. Request a "Student Observation Update" to trigger the automated email drafting workflow.
3. Follow the guided prompts to provide student initials, time, and behavior for a ready-to-send draft.
