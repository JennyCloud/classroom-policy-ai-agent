# Classroom Policy AI Agent 🤖

## 🎯 Project Goal
A proof-of-concept AI agent designed to assist Education Assistants (EAs) in BC schools. This agent quickly retrieves information from long school handbooks regarding safety, classroom conduct, and administrative procedures.

## 🛠️ Technology Stack
* **Platform:** Microsoft Copilot Studio (Agentic AI)
* **Knowledge Base:** BC School Policy Manual
* **Language:** English (Natural Language Processing)
